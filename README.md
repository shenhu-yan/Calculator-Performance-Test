# 计算器Web应用 - 性能测试与自动化测试

## 项目简介

这是一个基于HTML/CSS/JavaScript开发的简单计算器Web应用，包含完整的性能测试和自动化测试脚本。

## 功能特性

- ✅ 基本四则运算（加、减、乘、除）
- ✅ 小数运算支持
- ✅ 清除和删除功能
- ✅ 键盘输入支持
- ✅ 响应式设计
- ✅ 美观的UI界面

## 文件说明

```
├── calculator.html                 # 计算器Web应用主页面
├── calculator_performance_test.jmx # JMeter性能测试脚本
├── calculator_selenium_test.py     # Selenium自动化测试脚本
└── README.md                       # 项目说明文档
```

## 快速开始

### 运行计算器应用

直接双击打开 `calculator.html` 文件即可在浏览器中运行。

### 性能测试 (JMeter)

1. 安装 [Apache JMeter](https://jmeter.apache.org/)
2. 打开JMeter，导入 `calculator_performance_test.jmx` 文件
3. 配置服务器地址（默认：localhost:8080）
4. 运行测试计划

**性能测试配置：**
- 并发用户数：10
- 循环次数：100次/用户
- Ramp-Up时间：5秒
- 总请求数：1000次

### 自动化测试 (Selenium)

```bash
# 安装依赖
pip install selenium webdriver-manager

# 运行测试
python calculator_selenium_test.py
```

**测试覆盖场景：**
- 页面标题验证
- UI元素存在性检查
- 加减乘除运算测试
- 小数运算测试
- 清除功能测试
- 删除功能测试
- 复杂表达式计算
- 除零错误处理

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **性能测试**: Apache JMeter 5.6
- **自动化测试**: Python + Selenium WebDriver
- **版本控制**: Git

## 项目结构

### 计算器功能模块

- 数字输入 (0-9)
- 运算符 (+, -, ×, ÷)
- 等号 (=) - 执行计算
- 清除 (C) - 重置所有输入
- 删除 (←) - 删除最后一位
- 小数点 (.) - 输入小数

### 测试脚本详情

#### JMeter性能测试脚本

包含以下组件：
- HTTP请求默认值配置
- 线程组（模拟多用户并发）
- HTTP采样器（访问计算器页面）
- 响应断言（检查点）
- 监听器（结果收集）

#### Selenium自动化测试脚本

包含12个测试用例：
1. 验证页面标题
2. 验证显示屏存在
3. 验证所有按钮存在
4. 加法运算测试 (5+3=8)
5. 减法运算测试 (9-4=5)
6. 乘法运算测试 (6×7=42)
7. 除法运算测试 (15÷3=5)
8. 小数运算测试 (2.5+1.5=4)
9. 清除功能测试
10. 删除功能测试
11. 复杂运算测试 ((10+5)×2-8=12)
12. 除零错误处理测试

## 性能指标

通过JMeter可以获取以下性能数据：
- 平均响应时间
- 吞吐量 (TPS)
- 错误率
- 响应时间百分位 (90%, 95%, 99%)
- 并发用户数 vs 响应时间关系图
